# WebUI and Configuration

Default config path:

- `$OMNIMEM_HOME/omnimem.config.json`
- fallback: `~/.omnimem/omnimem.config.json`

Start app:

```bash
~/.omnimem/bin/omnimem
```

WebUI provides:

- Status & actions tab
- Configuration tab
- Memory browser tab
- Daemon toggle and bootstrap sync action
